# Bucket 8: Domain Deep Dive — Hardware-Adjacent Frontend

Method: WebSearch + Chrome | ~20-30 min | Manual trigger only

Candidate has ХНУРЭ/ХРТТ education (radio engineering, electronics). This source targets frontend roles at companies where that background is a competitive advantage: EDA tools, robotics UI, IoT dashboards, telecom/RF platforms, embedded device management, PCB/schematic viewers.

## Why this source exists

Most frontend engineers competing for these roles have CS backgrounds. A candidate with radio engineering education + React/TypeScript skills is a rare combo — especially for companies building UIs that require understanding of circuits, signals, RF, protocols, or hardware workflows.
