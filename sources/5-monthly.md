# Bucket 5: Monthly Research

Method: Chrome | ~20-30 min
Run on 1st of each month or when explicitly requested.

| Source | Priority | Action |
|--------|----------|--------|
| HN Who's Hiring | **highest** | news.ycombinator.com → find current month's "Who is hiring?" thread (posted 1st). Search comments for: React, TypeScript, frontend, dashboard, visualization, observability, Canada, remote, B2B, devtools, fintech, compliance, healthcare, logistics, "supply chain", "internal tools", "admin", "human-in-the-loop", "AI tools", annotation, labeling, workflow, automation, "no-code", orchestration. Process thoroughly — highest-signal monthly source. |
| Greylock Portfolio | normal | jobs.greylock.com → search "frontend" "React" → check talent network for new matches |
| G2i | normal | g2i.co → check for new React-specific opportunities (contract + full-time) |
| Levels.fyi Research | normal | levels.fyi → check "Still Hiring" tracker for target companies → review salary data for Grafana, Mercury, Ramp, Stripe, Brex in Canada → save to report for negotiation prep |
| Glassdoor Canada | normal | glassdoor.ca → check interview questions and salary data for companies with upcoming interviews. Research source only. |
| Underdog.io | normal | If account active: check weekly batch matches for React/frontend at curated startups |
