# Bucket 7: Staffing Agencies

Method: Chrome | ~15-20 min

## How This Differs from Target Companies (Bucket 3)

Agencies are NOT like companies. The primary action is **submit resume / create profile**, not search for specific postings. An agency recruiter matches you to contracts — you don't need to find the vacancy yourself.

Some agencies also have a job board. If they do, browse it as a secondary action after registration.

## Actions Per Agency

1. **Register / submit resume** — create a profile if new, or update resume if already registered.
2. **Browse job board** (if available) — scan for relevant frontend/React roles using title variants from `data/title-variants.md`.
3. **Note status** — track whether profile is active, resume is current, and last contact date.
